# Deprecation Notice: Artificial Intelligence API

The RingCentral Artificial Intelligence API listed below has been deprecated and will no longer receive updates, enhancements, or new feature support.

- `/ai/insights/v1/async/analyze-interaction`
- `/ai/audio/v1/async/speaker-diarize`
- `/ai/audio/v1/async/speech-to-text`
- `/ai/text/v1/async/summarize`
- `/ai/text/v1/async/punctuate`
- `/ai/status/v1/jobs/jobId`

Developers should transition to the [AI Conversation Expert API](./ace), which is the recommended replacement for accessing conversation insights and advanced AI-driven capabilities. The AI Conversation Expert API provides improved functionality, enhanced accuracy, and ongoing support for features such as conversation analysis, insights generation, and other AI-powered capabilities.

## What this means for developers

- Existing integrations using the Artificial Intelligence API may continue to function temporarily but are not guaranteed to be supported long term.
- No new features or improvements will be added to the deprecated API.
- Bug fixes and support for the deprecated API may be limited or discontinued.
