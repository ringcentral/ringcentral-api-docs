Detailed Description

The RcvAudioController class is a managing class to control the audio-related functions and states in an active meeting session, such as mute or unmute local or remote audio.

Method Documentation

◆ disableAudio

- (int64_t) disableAudio

Disables the audio session.

A successful call of disableAudio triggers the onLocalAudioStreamStateChanged event callback to the local user. For the local track, the client SDK will stop capturing frames from the audio source and not sending any audio data to the server. For the remote track, the server will not forward any audio streams to the local user.

Returns
0 means the action succeeds or fails otherwise

◆ enableAudio

- (int64_t) enableAudio

Enables the audio session.

A successful call of enableAudio triggers the onLocalAudioStreamStateChanged event callback to the local user. The server will start to forward the remote audio streams to the local user.
