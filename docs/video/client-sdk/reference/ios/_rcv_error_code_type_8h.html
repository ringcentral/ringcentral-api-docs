enum RcvErrorCodeType : NSInteger

Global error code for RCV SDK

Enumerator
RcvErrorCodeTypeErrOk 

No Errors

RcvErrorCodeTypeErrFailed 

A general error occurs (no specified reason).

RcvErrorCodeTypeErrInvalidArgument 

An invalid parameter is used. For example, the specific meeting ID includes illegal characters.

RcvErrorCodeTypeErrNotReady 

The SDK module is not ready. Possible solutions:

  • Check the audio device.
  • Check the completeness of the application.
  • Re-initialize the RCV engine.
RcvErrorCodeTypeErrNotSupported 

The SDK does not support this function.

RcvErrorCodeTypeErrRefused 

The request is rejected.

RcvErrorCodeTypeErrNotInitialized 

The SDK is not initialized before calling this method.

RcvErrorCodeTypeErrNoPermission 

No permission exists. Check if the user has granted access to the audio or video device.

RcvErrorCodeTypeErrJoinMeetingRejected 

The request to join the meeting is rejected.

  • This error usually occurs when the user is already in the meeting, and still calls the method to join the meeting, for example, RcvEngine::joinMeeting
  • The user tries to join the meeting with a token that is expired.
RcvErrorCodeTypeErrLeaveMeetingRejected 

The request to leave the meeting is rejected. This error usually occurs:

  • When the user has left the meeting and still calls RcvEngine::leaveMeeting to leave the meeting. In this case, stop calling RcvEngine::leaveMeeting.
  • When the user has not joined the meeting and still calls RcvEngine::leaveMeeting to leave the meeting. In this case, no extra operation is needed.
RcvErrorCodeTypeErrAborted 

The SDK gives up the request due to too many requests.

RcvErrorCodeTypeErrResourceLimited 

The application uses too much of the system resources and the SDK fails to allocate the resources.

RcvErrorCodeTypeErrInvalidMeetingId 

The specified meeting ID is invalid. Please try to rejoin the meeting with a valid meeting ID.

RcvErrorCodeTypeErrTokenExpired 

The token expired due to one of the following reasons:

  • Authorized Timestamp expired: The timestamp is represented by the number of seconds elapsed since 1/1/1970. The user can use the Token to access the RCV SDK within 24 hours after the Token is generated. If the user does not access the RCV SDK after 24 hours, this Token is no longer valid.
RcvErrorCodeTypeErrInvalidToken 

The token is invalid due to one of the following reasons:

  • The App Certificate for the project is enabled in Console, but the user is still using the App ID. Once the App Certificate is enabled, the user must use a token.
  • The uid is mandatory, and users must set the same uid as the one set in the RcvEngine::joinMeeting method.
RcvErrorCodeTypeErrInternalError 

SDK internal error

RcvErrorCodeTypeErrNeedPassword 

The meeting is encrypted and requires a password. Please re-join with password

RcvErrorCodeTypeErrInWaitingRoom 

You are in the waiting room of the meeting, waiting for the meeting host to admit you to join

RcvErrorCodeTypeErrBreakoutRoomTransition 

You're making a transition between breakout rooms

RcvErrorCodeTypeErrWaitingHostJoinFirst 

Waiting for the meeting host to join the meeting first

RcvErrorCodeTypeErrAnotherMeetingAlreadyInProgress 

There is already a meeting instance in progress

RcvErrorCodeTypeErrMovedToWaitingRoom 

You are moved from meeting room to waiting room

RcvErrorCodeTypeErrDeniedFromWaitingRoom 

You are denied to join

RcvErrorCodeTypeErrMeetingAlreadyLocked 

The meeting already locked

RcvErrorCodeTypeErrMeetingAlreadyUnlocked 

meeting already unlocked

RcvErrorCodeTypeErrSharingAlreadyLocked 

The sharing already locked

RcvErrorCodeTypeErrSharingAlreadyUnlocked 

The sharing already unlocked

RcvErrorCodeTypeErrNoMicrophonePermission 

No microphone permission exists

RcvErrorCodeTypeErrNoCameraPermission 

No camera permission exists

RcvErrorCodeTypeErrCameraIsMuted 

The camera is muted

RcvErrorCodeTypeErrToRecreateMeetingChat 

Failed to create chat and recreated in an active meeting

RcvErrorCodeTypeErrCreateMeetingChatFailed 

Create meeting chat failed in an active meeting

RcvErrorCodeTypeErrSendMeetingChatMessageFailed 

Failed to send chat message in an active meeting

A general error occurs when updated personal meeting settings or scheduled meetings
