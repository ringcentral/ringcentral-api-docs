Detailed Description

The RcvMeetingUserController class is a managing class to control the meeting user related functions and states in an active meeting, such as admitting or denying the participants who are in the waiting room and assigning moderators, etc.

Method Documentation

◆ admitAll

- (int64_t) admitAll

Admits all currently in the waiting room users to join the meeting.

A successful call of admitAll triggers the onMeetingJoin event callback for the waiting room users, and the remote user will retrieve the onUserJoin event notification.

Note
Only the meeting host or moderator has permission to invoke this method.
Returns
0 means the action succeeds or fails otherwise
