Detailed Description

The RcvVideoDeviceManager class is a managing class to manage the video devices and test them out.

◆ registerEventHandler:

- (void) registerEventHandler: (nullable id< RcvVideoDeviceEventHandler >)  handler

Registers the event handler.

The RingCentral video client SDK uses the RcvVideoDeviceEventHandler interface class to send video device relevant event callbacks to the application, and the application inherits the methods of this interface class to retrieve those event callbacks.

◆ startDeviceTest:

- (int64_t) startDeviceTest: (int64_t)  view

Starts the camera test.

A successful call of startDeviceTest triggers capturing the video frames from the camera and displaying them in the video view. If the user can see their video that means the camera works properly.

Note
Suggests calling this method before joining a meeting to make sure the camera works properly in the meeting.
Parameters
viewThe video view is used to display the video frames.
Returns
0 means success or fails otherwise
