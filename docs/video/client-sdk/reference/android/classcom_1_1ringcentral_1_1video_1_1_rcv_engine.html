Detailed Description

The RcvEngine class is the entry point of the RingCentral video client SDK that empowers the applications to easily and quickly build real-time audio and video communication.

RcvEngine is the base interface class of the RCV SDK that provides the main RCV SDK methods invoked by your application. Enable the RCV SDK's communication functionality through the RcvEngine instance, then call the methods of this object.

Joins a meeting with a specific meeting id.

A successful call of joinMeeting triggers the onMeetingJoin event callback for the local user, and the remote user will retrieve the onUserJoined event callback.

Note
If the user is the meeting host, by entering the meeting, the user will be granted the host role automatically. If the user is a participant and is not allowed Join Before Host or the Waiting Room option has been enabled, they have to wait until the meeting starts or the host admits.
